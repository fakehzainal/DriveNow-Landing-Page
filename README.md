# DriveNow - Landing Page Rental Mobil

Landing page website persewaan mobil dengan Next.js App Router, Tailwind CSS, dan komponen gaya shadcn/ui.

## Menjalankan Project

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.
